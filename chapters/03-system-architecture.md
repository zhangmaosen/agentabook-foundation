# 03 System Architecture

Agentabook can be understood as three connected layers.

## GitHub Layer

Internal collaboration for execution agents.

## DataHub Layer

Internal governance for know-how, context, and dependencies.

## TransactionHub Layer

Inter-organization coordination, exchange, and settlement.

The first two layers are primarily intra-organizational.
The third layer is primarily inter-organizational.

This separation is important because internal operating quality should exist before external market liquidity.
